# FontFeels

Type anything. See it rendered in 24 fonts instantly. Find the one that feels right.

## Live Demo

🔗 [jasnoor-kaur1501.github.io/FontFeels](https://jasnoor-kaur1501.github.io/FontFeels)

## What it does

- Type any text in the search bar — all 24 fonts update in real time
- Adjust font size from 20px to 80px with a slider
- Filter by category: sans-serif, serif, display, handwritten, monospace
- Every font has a vibe description — not just "serif", but what it actually feels like
- Save favourites, copy font names with one click
- Copy your preview text to clipboard

## Font categories

| Category | Fonts |
|----------|-------|
| Display | Abril Fatface, Bebas Neue, Alfa Slab One, Lobster, Monoton, Righteous |
| Serif | DM Serif Display, Italiana, Libre Baskerville, Playfair Display, Yeseva One |
| Sans-serif | Exo 2, Josefin Sans, Raleway, Secular One, Space Grotesk |
| Handwritten | Architects Daughter, Caveat, Dancing Script, Gochi Hand, Pacifico, Permanent Marker, Satisfy |
| Monospace | IBM Plex Mono |

## Design

Space Grotesk UI · Pure white, black ink, minimal borders · Type foundry / design tool aesthetic. Completely different from Days 01–04.

## Tech

HTML · CSS · Vanilla JS · Google Fonts · No API · No dependencies · GitHub Pages

## Run locally

Download or clone → open `index.html` in any browser. Done.
